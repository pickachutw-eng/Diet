個人熱量計算

限定使用 Google 帳號登入。登入後，餐點、每日目標與模板會依照 Firebase Auth 的 UID 存進 Realtime Database。

使用前請到 Firebase Console 啟用 Authentication → Sign-in method → Google，並建立 Realtime Database。程式碼中的 firebaseConfig 必須換成你自己的 Web App config。

新增餐點

新增餐點會記錄在目前查看的日期；營養欄位請填 1 份的數值，送出時會依「份量倍數」換算。蛋白質、碳水、脂肪三項填完後，熱量會自動以 P×4 + C×4 + F×9 計算。

常用餐點模板

模板不再預設內容；請新增自己的常用餐點。模板數值也以 1 份為單位，直接新增時會依份量倍數換算。
